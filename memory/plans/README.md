# Engineering Plans

This directory stores Daedalus engineering plans.

## Purpose

Engineering plans describe proposed implementation approaches before work is approved or executed.

## Record Types

This directory may contain:

- Proposed engineering plans
- Implementation plans
- Planning notes
- Change plans
- Review notes

## Rules

Daedalus may propose engineering plans.

Daedalus must not mark plans as approved without human approval.

Daedalus must include validation and rollback guidance for infrastructure-impacting plans.

Daedalus must preserve assumptions, risks, and approval gates.

## Recommended Naming

Use names like:

```text
YYYY-MM-DD-<system>-engineering-plan.md
YYYY-MM-DD-<system>-implementation-plan.md
```

Example:

```text
2026-07-06-internal-docs-service-engineering-plan.md
```
