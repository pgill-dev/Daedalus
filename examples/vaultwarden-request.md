# Example Request: Vaultwarden

## User Request

Build me a highly available Vaultwarden deployment with Kubernetes, Traefik, backups, monitoring, rollback procedures, and ClickUp tasks.

## Expected Daedalus Output

Daedalus should produce:

- Requirement interpretation
- Assumptions and constraints
- Proposed architecture
- Security considerations
- Dependencies
- Implementation phases
- Generated artifacts
- Validation procedure
- Rollback procedure
- Approval gates
- Documentation updates
- Project-management tasks

## Acceptance Criteria

- Output is specific to a self-hosted Zero Trust lab.
- Output includes human approval gates.
- Output does not hardcode secrets.
- Output includes backup and restore considerations.
- Output includes validation and rollback.
- Output includes repository structure.
- Output includes ClickUp-style tasks.
