# CLI Examples

## Purpose

This directory contains examples showing how to use the Daedalus CLI to create proposed engineering artifacts.

## Examples

| File | Purpose |
|---|---|
| `create-engineering-request.md` | Example for creating a new engineering request |
| `create-adr.md` | Example for creating a new ADR |
| `create-validation-and-rollback.md` | Example for creating validation and rollback artifacts |

## Safety Model

The CLI generates local Markdown artifacts only.

The CLI does not apply infrastructure changes.

All generated artifacts are proposed until reviewed and approved by a human.
